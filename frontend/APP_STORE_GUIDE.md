# Guide de Soumission App Store - Latence

## Prérequis

1. **Compte Apple Developer** (99€/an) : https://developer.apple.com
2. **Xcode** installé sur un Mac
3. **Node.js** et **EAS CLI** installés

## Étape 1 : Configuration EAS

### 1.1 Installation d'EAS CLI
```bash
npm install -g eas-cli
```

### 1.2 Connexion à Expo
```bash
eas login
```

### 1.3 Configuration du projet
```bash
cd frontend
eas build:configure
```
Cela va créer un `projectId` que vous devrez copier dans `app.json` (remplacer `YOUR_EAS_PROJECT_ID`).

## Étape 2 : Configuration dans app.json

Modifiez ces valeurs dans `/frontend/app.json` :

```json
{
  "expo": {
    "owner": "VOTRE_USERNAME_EXPO",
    "extra": {
      "eas": {
        "projectId": "ID_RETOURNÉ_PAR_EAS"
      }
    },
    "updates": {
      "url": "https://u.expo.dev/ID_RETOURNÉ_PAR_EAS"
    }
  }
}
```

## Étape 3 : Configuration dans eas.json

Modifiez ces valeurs dans `/frontend/eas.json` :

```json
{
  "submit": {
    "production": {
      "ios": {
        "appleId": "votre@email.apple.com",
        "ascAppId": "ID_APP_STORE_CONNECT",
        "appleTeamId": "VOTRE_TEAM_ID"
      }
    }
  }
}
```

### Comment trouver ces IDs :
- **appleId** : Votre email Apple Developer
- **appleTeamId** : App Store Connect → Utilisateurs et accès → ID de l'équipe
- **ascAppId** : Créez d'abord l'app dans App Store Connect, puis copiez l'ID

## Étape 4 : Build iOS Production

```bash
cd frontend
eas build --platform ios --profile production
```

Le build prend environ 15-30 minutes. Vous recevrez un lien vers le fichier `.ipa`.

## Étape 5 : Soumission à l'App Store

### Option A : Soumission automatique
```bash
eas submit --platform ios
```

### Option B : Soumission manuelle
1. Téléchargez le `.ipa` depuis le lien EAS
2. Ouvrez **Transporter** sur Mac
3. Glissez le `.ipa` dans Transporter
4. Cliquez sur "Livrer"

## Étape 6 : App Store Connect

### 6.1 Création de l'App
1. Allez sur https://appstoreconnect.apple.com
2. Mes apps → + → Nouvelle app
3. Remplissez :
   - **Nom** : Latence
   - **Langue principale** : Français
   - **Bundle ID** : com.atelierbenamer.latence
   - **SKU** : latence-journal-2026
