# Latence - Journal Immersif Astrologique

## Vision
Application mobile de journal intime avec une forte orientation astrologie, bien-être émotionnel et IA omniprésente.

## Fonctionnalités Implémentées

### 1. Thème Chaleureux & Cosy ✅
- **Light Theme**: Tons beige/crème chauds, ambiance bougie
- **Dark Theme**: Brun foncé chaleureux, lueur de bougie
- **Silence Theme**: Noir profond avec touches chaudes

### 2. Animations & Ambiance ✅ (NEW)
- **Bougies animées** (CandleFlame): Flammes qui vacillent dans les headers
- **Étoiles scintillantes** (TwinklingStars): Fond étoilé pour les modes sombres

### 3. Module Cosmos (ex-Astres) ✅
- Onglets améliorés avec icônes: Mon Profil, Lune, Celtique, Arabe, Maisons
- Bougies de chaque côté du titre

### 4. Profil Astrologique Complet ✅
- **Signe Solaire**: Avec élément et planète
- **Signe Lunaire**: Calcul précis + description détaillée
- **Arbre Celtique**: 13 arbres avec personnalité complète
- **Demeure Arabe**: 28 demeures avec descriptions

### 5. Mode Méditation ✅ (NEW)
- Respiration guidée avec 4 patterns:
  - Calme (4-4-6-2)
  - Énergisant (4-0-4)
  - Sommeil (4-7-8)
  - Focus (4-4-4-4)
- Flamme de bougie animée au centre
- Timer et compteur de respirations

### 6. Compatibilité Amoureuse ✅ (NEW)
- Compare deux profils astrologiques
- Score d'harmonie cosmique
- Forces, défis et conseils personnalisés

### 7. Statistiques Émotionnelles ✅ (NEW)
- Distribution des humeurs
- Streak (jours de suite)
- Énergie moyenne
- Humeur dominante
- Historique de la semaine

### 8. Modules IA ✅
- **Écrire & Sceller**: Journal avec interprétation IA
- **Carnet des Rêves**: Log et interprétation des rêves
- **IA Miroir**: 3 modes (Réflexion, Analyse, Questionnement)
- **Dialogue Intérieur**: Compagnon poétique IA

### 9. Aura Avatar ✅
- Avatar pulsant reflétant l'humeur
- Intégré au header et à la page profil

## Architecture Technique

### Frontend
- React Native + Expo (Router)
- TypeScript
- React Native Reanimated
- Composants: CandleFlame, TwinklingStars, AuraAvatar

### Backend  
- FastAPI (Python)
- MongoDB
- GPT-4o via emergentintegrations
